Internet & Browsers

A browser has many functions but the basic ones are to request, send and display data. We send a data package containting something like "google.com" out to a DNS Server, which responds with an ip adress which should lead to a specific server. the browser then tries to connect to that ip adress and asks it for some content.

If everything went fine, this response comes in the form of HTML code, possibly CSS & JavaScript and a status code. The point of the status code is to identify errors that may have happened. Here's a list of HTTP Status Codes! It then the browsers job to interpret all of this code and display it for the viewer.
