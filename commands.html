Tech & Commands

Git & Github

One of the most convenient technologies i use is Git. Git is basically what saves & quick saves are for games. except it saves code instead and then stores it somewhere accessible for you or even others people. This allows you or others to work on different machines but still write code for the same program or website.

Commands

Keybind Function
Ctrl + ← OR → Jumps the marker 1 word in the direction you click.
Shift Selects all the code your selector moves across
Home OR End Jumps the marker to the Begining OR End of selected line.
Pg up OR pg down Jumps the marker to the Top OR Bottom of the file.
Tab OR Shift + Tab Jumps selected code 1 indentation to the right or left
Alt + Tab Jumps between programs
